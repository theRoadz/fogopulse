# Story 2.5.1: Price Chart Time Scale Enhancement

Status: done

## Story

As a trader,
I want the price chart to display a fixed 60-second window with 8-10 second tick intervals,
So that I can clearly see recent price movements with consistent time labels.

## Acceptance Criteria

1. **Given** the trading page with price chart, **When** viewing the chart, **Then** exactly 60 seconds of price history is visible at once
2. X-axis labels are spaced approximately 8-10 seconds apart (e.g., 9:59:20, 9:59:28, 9:59:36...)
3. Y-axis automatically scales to fit the price range within the visible 60-second window
4. New data points appear on the right edge, older data scrolls left
5. Full price line is always visible within the time window (no clipping at edges)
6. Chart scrolls automatically to keep the latest price visible at the right edge
7. Layout split is 70% chart / 30% trade ticket on desktop (matching Polymarket)
8. Chart height is fixed at 280px (mobile) / 300px (tablet+) - compact like Polymarket

### Lightweight Charts Time Scale API

Key methods:
- `timeScale.setVisibleRange({ from, to })` - Set exact visible time range
- `timeScale.scrollToRealTime()` - Scroll to show latest data
- `fixedRightEdge: true` - Keeps right edge fixed at latest data point

### Updated Implementation

```typescript
// Time scale configuration constants (add near top of file)
const VISIBLE_RANGE_SECONDS = 60 // 1 minute visible window

// Updated timeScale config
timeScale: {
  borderColor: colors.borderColor,
  timeVisible: true,
  secondsVisible: true,
  fixedRightEdge: true, // Keep right edge at latest data
  tickMarkFormatter: (time: UTCTimestamp) => {
    const date = new Date(time * 1000)
    const hours = date.getHours()
    const minutes = date.getMinutes().toString().padStart(2, '0')
    const seconds = date.getSeconds().toString().padStart(2, '0')
    const period = hours >= 12 ? 'PM' : 'AM'
    const displayHours = hours % 12 || 12
    return `${displayHours}:${minutes}:${seconds} ${period}`
  },
},
```

### scrollToRight Helper Function

```typescript
// Add inside component, before useEffects
const scrollToRight = useCallback(() => {
  if (!chartRef.current || data.length === 0) return
  const timeScale = chartRef.current.timeScale()
  const lastTime = data[data.length - 1].time as number
  const rangeStart = lastTime - VISIBLE_RANGE_SECONDS
  timeScale.setVisibleRange({
    from: rangeStart as UTCTimestamp,
    to: lastTime as UTCTimestamp,
  })
}, [data])
```
